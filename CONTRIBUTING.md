# Contributing

Thanks for your interest in MadeOnShroom! 🍄

This repository is a **public information repository** — it hosts the project overview, whitepaper,
and high-level architecture docs. The application source code is maintained privately, so we are not
accepting code pull requests here.

You can still help:

## Report a bug

Open a [bug report](https://github.com/madeonshroom) issue with:

- What you expected to happen vs. what actually happened
- Steps to reproduce
- The page/URL, and if relevant the transaction signature or token address
- Screenshots if helpful

## Request a feature

Open a feature request issue describing the idea and the problem it solves.

## Report a security issue

**Do not** open a public issue for security problems. Follow [SECURITY.md](./SECURITY.md) and email
us privately instead.

## Improve the docs

Spotted a typo or an unclear explanation in the README, whitepaper, or architecture doc? A small pull
request against these Markdown files is welcome.
